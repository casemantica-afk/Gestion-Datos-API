#API de Gestión de Inventario - Django REST Framework

Este proyecto es una API RESTful diseñada para la administración de productos y control de inventario. Es el primer pilar de mi portafolio como desarrollador Backend.

##Tecnologías Utilizadas
* Python 3.x: Lenguaje principal.
* Django 6.0: Framework web.
* Django REST Framework (DRF): Para la construcción de la API.
* SQLite: Base de datos utilizada.
* Git/GitHub: Control de versiones.

## Características Técnicas
- CRUD Completo: Endpoints preparados para Crear, Leer, Actualizar y Eliminar productos.
- Serialización de Datos: Conversión de modelos de base de datos a formato JSON.
  .
